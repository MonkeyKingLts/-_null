# 电商用户评论情感倾向分析平台

本科毕业设计示例项目。在四个可选题目中，本题**开发难度最低、演示效果最好**：无需摄像头、录音设备或第三方 Agent 平台，使用开源中文 NLP 库即可在本地完整运行。

## 选题说明

| 题目 | 难度 | 说明 |
|-----|------|------|
| **电商评论情感分析** | ★★☆ | **推荐**，SnowNLP + Web 即可 |
| 录音转写应用 | ★★★ | 依赖语音模型/API |
| 扣子平台 Agent | ★★★ | 依赖外部平台与账号 |
| 人脸识别系统 | ★★★★ | 需摄像头与 CV 模型 |

## 功能列表

- 单条评论情感分析（得分 + 正/中/负 + 关键词）
- 批量粘贴导入、CSV 文件导入
- 数据概览仪表盘（统计卡片 + 饼图/柱状图/折线图）
- 评论列表筛选、删除、清空
- 关键词频次统计（支持按情感类别）
- 分析结果 CSV 导出
- 内置 30 条电商评论示例数据

## 技术栈

- **后端：** Python 3.10+、FastAPI、SQLAlchemy、SQLite
- **NLP：** SnowNLP（情感分析）、jieba（分词）
- **前端：** HTML、CSS、JavaScript、Chart.js

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动服务
python run.py

# 3. 浏览器访问
# http://localhost:8000
# API 文档：http://localhost:8000/docs
```

## 项目结构

```
backend/          # FastAPI 后端与情感分析模块
frontend/         # 单页 Web 前端
data/             # SQLite 数据库与样例 CSV
docs/             # 毕业设计说明书（Markdown）
run.py            # 启动入口
```
