关键词说明

系统使用 jieba 对评论进行中文分词，并统计去除停用词后的高频词。 通过对比正面与负面评论关键词，可以帮助运营人员快速了解用户关注点， 例如“续航”“做工”“客服”等维度。

情感得分由 SnowNLP 计算，范围 0 到 1。得分 ≥ 0.6 判定为正面， ≤ 0.4 判定为负面，其余为中性。
